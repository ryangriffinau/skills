Make this writing unsummarizable: $ARGUMENTS

The target: leave so little fluff that removing any words — which is all a summary
does — costs real ideas. Every sentence should carry information found nowhere else
in the piece.

- If a sentence survives deletion with no loss of meaning, delete it.
- Replace abstractions with the specific thing; replace adjectives with facts.
- Fuse sentences that share a subject. One idea per clause, no idea twice.
- Keep the voice and the interesting ideas; raise the density. Go shorter only when
  nothing interesting is lost.

Density, not brevity, is the goal — never sand away an idea to hit a word count.
Return the tightened version on its own. If a passage is already maximally dense,
say so and leave it.
